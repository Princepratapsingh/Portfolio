# Prince Pratap Singh - Portfolio Website

## Description
A modern, responsive portfolio website showcasing my skills, projects, and experience as a developer. Features a coding-themed design with interactive animations and a functional contact form.

## Features
- **Responsive Design**: Optimized for both mobile and desktop viewing
- **Interactive Background**: Matrix-style code rain animation
- **Contact Form**: Functional form that sends emails via mailto protocol
- **Social Links**: Direct links to LinkedIn, GitHub, and email in the footer
- **Smooth Navigation**: Sticky navigation with mobile hamburger menu
- **Project Showcase**: Grid layout displaying key projects with technologies used

## Technologies Used
- **HTML5**: Semantic structure and content
- **CSS3**: Custom styling, animations, and responsive design with Flexbox/Grid
- **JavaScript (ES6+)**: Interactive features including form handling and animations

## How to View Locally
1. Clone or download the repository
2. Open `index.html` in your web browser
3. The website will load with all interactive features

## Deployment
This is a static website that can be deployed on any hosting service. For GitHub Pages:

1. Push the code to a GitHub repository
2. Go to repository Settings > Pages
3. Select "Deploy from a branch" and choose the main branch
4. The site will be live at `https://yourusername.github.io/repository-name/`

## Sections
- **Hero**: Introduction with animated background
- **About**: Personal summary and background
- **Projects**: Showcase of key projects with links and tech stacks
- **Skills**: Categorized technical skills
- **Contact**: Contact form and additional information
- **Footer**: Social media links and copyright
